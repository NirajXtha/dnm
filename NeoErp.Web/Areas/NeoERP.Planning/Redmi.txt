In NeoERP.Web/Web.config, place a setting key below for product limit
  <appSettings>
    <add key="ProductSelectionLimit" value="50"/>
  </appSettings>

Item filter condition are maintained by ProductCondition.xml file
file path are used as HttpContext.Current.Server.MapPath("~/Areas/NeoERP.Planning/ProductCondition.xml");

<ProductCondition>
  <Vendor ID="01" Name="HIM">
    <ConditionQuery> AND CATEGORY_CODE NOT IN ('SP','RM','SC','FA') </ConditionQuery>
  </Vendor>
</ProductCondition>

For different Database or Vendor Place Where condition  value inside <ConditionQuery></ConditionQuery> under <Vendor ID="01" Name="As your desire">.
